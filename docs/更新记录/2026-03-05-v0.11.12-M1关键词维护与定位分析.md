## 三、导入表格规则（M1）

支持列（列名中英都可识别）：
- `关键词` / `keyword`（必填）
- `同义词` / `synonyms`（可选，逗号/分号分隔）
- `类型` / `type`（建议 `agreement`）
- `关键项` / `keySection`（如 `ppm`、`inspectionRequirements`）
- `机型` / `machineType`（可选）
- `工位` / `station`（可选）
- `权重` / `weight`（可选，默认 `1`）
- `启用` / `enabled`（可选：`是/否`、`1/0`）
- `备注` / `remark`（可选）
- `优先级` / `priority`（可选，默认 `1`）

说明：
- 若同一行含 `机型/工位/关键项`，导入时会同时创建绑定记录。
- 关键词按 `keyword + type + key_section` 去重并更新。
