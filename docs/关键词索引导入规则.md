# 关键词索引导入规则（M1）

本规则用于“资料库管理 -> 关键词索引”页的表格导入。

## 一、支持格式
- `.xlsx`
- `.xls`
- `.csv`

## 二、推荐模板（小白版，优先）

| 列名 | 必填 | 说明 |
|---|---|---|
| 机型 | 是 | 如 `数码卷绕设备-绿胶线扫相机` |
| 工位 | 是 | 可写 `ALL` |
| 解析关键项 | 是 | 如：产品尺寸/PPM/工位要求/检测精度 |
| 关键词引词 | 是 | 多个词可用 `、` `,` `;` 分隔 |
| 关联项 | 否 | 如：`产品尺寸、工位要求` |
| 项目实际解析的内容简介（备注） | 否 | 自由描述，便于后续总结 |

> 系统会自动将小白版字段映射为关键词、同义词、关键项、绑定关系与备注。

## 三、高级模板列定义（可选）
| 列名（中文） | 列名（英文） | 必填 | 说明 |
|---|---|---|---|
| 关键词 | keyword | 是 | 主关键词，如 `PPM` |
| 同义词 | synonyms | 否 | 多个词可用 `,` `，` `;` `；` 分隔 |
| 类型 | type | 否 | 建议 `agreement`（也支持 `vision` `cost` `all`） |
| 关键项 | keySection / key_section | 否 | 例：`ppm`、`inspectionRequirements` |
| 机型 | machineType / machine_type | 否 | 若填写可自动创建绑定 |
| 工位 | station | 否 | 若填写可自动创建绑定 |
| 权重 | weight | 否 | 默认 `1` |
| 启用 | enabled | 否 | `是/否`、`1/0`、`true/false` |
| 备注 | remark | 否 | 说明信息 |
| 优先级 | priority | 否 | 绑定优先级，默认 `1` |

## 四、关键项可选值（建议）
- `productSize`
- `ppm`
- `inspectionRequirements`
- `inspectionObjectSpecs`
- `inspectionPrecision`
- `stationRequirements`
- `brandRequirements`
- `industrialComputerRequirements`
- `softwareRequirements`

## 五、导入行为
- 关键词按 `keyword + type + key_section` 去重。
- 若重复，将更新同义词/权重/启用/备注等字段。
- 同一行若包含 `机型` 或 `工位` 或 `关键项`，会自动创建绑定记录。

## 六、示例（CSV）
```csv
机型,工位,解析关键项,关键词引词,关联项,项目实际解析的内容简介（备注）
数码卷绕设备-绿胶线扫相机,ALL,产品尺寸,工艺、产品尺寸、极片尺寸、极耳尺寸,,产品尺寸是否一致，是否有新的边界要求
数码卷绕设备-绿胶线扫相机,ALL,PPM,节拍、PPM,产品尺寸,设备生产节拍要求是多少，是否分工位不同
数码卷绕设备-绿胶线扫相机,模切后切口检测工位,检测精度,精度、重复精度、分辨率,,若协议无明确值可结合视野和分辨率推算
```
