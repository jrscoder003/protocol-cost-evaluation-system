# 资料库文档目录说明

本目录用于**统一保存维护中心相关资料库文档**，与系统页面中的资料库模块一一对应。

## 子目录说明

- `01_历史技术协议库`：历史客户技术协议（pdf/doc/docx）。
- `02_提供资料模板库`：客户模板、输出模板（pptx/xlsx/docx）。
- `03_信息文档知识库`：工艺说明、操作说明、标准规范等参考文档。
- `04_关注信息规则附件`：规则说明、判定依据附件文档。
- `05_导出归档（报告_报价_统计表）`：系统导出的基础设计报告、报价单、相机光源统计表归档。

## 命名建议

建议文件名包含：`项目号_客户_设备_版本_日期`。

示例：`JC250028_锂威_焊接卷绕一体机_协议_V1_20260302.pdf`
